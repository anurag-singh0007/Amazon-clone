# 🛒 Amazon Clone

A modern and responsive **Amazon Clone** built to replicate the core UI and functionality of the Amazon homepage. This project demonstrates strong fundamentals in frontend development and clean UI structuring.

## 📌 Features

- 🏠 Amazon-like homepage layout  
- 🔍 Search bar UI  
- 🛍️ Product listing section  
- 📱 Fully responsive design  
- 🧭 Navigation bar with categories  
- 🧾 Clean and structured UI components  

## 🛠️ Tech Stack

- **HTML5** – Structure  
- **CSS3** – Styling & layout  

## ⚙️ Installation & Setup

1. Clone the repository: git clone https://github.com/anurag-singh0007/Amazon-clone.git
   
2. Navigate to project folder: cd Amazon-clone
   
3. Open `index.html` in your browser
